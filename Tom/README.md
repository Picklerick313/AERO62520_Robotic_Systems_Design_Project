# Tom

Mechanical Design

This repository contains all mechanical design files for the payload sled, manipulator mount, camera housing, and peripheral structures used on the rover. The designs were created using Fusion360 software and optimised for 3D printing.

📁 CAD Files

All CAD models are located in the CAD/ directory.
This folder includes:

A full assembly model of the payload sled

Individual part files for each structural component

Manufacturing-ready exports for 3D printing

A compressed ZIP archive containing the complete CAD project for easy download

Users can extract the ZIP file to view, modify, or inspect the full design in the original CAD environment.

🛠 Manufacturing Notes

All printed components were designed with PLA in mind

Overhangs were minimised and fillets added to improve print reliability

Parts are oriented to maximise in-plane strength and reduce risk of layer separation

Assemblies are modular to allow rapid iteration, replacement, and testing

📐 Purpose of the Mechanical Design

The mechanical design supports:

Mounting the manipulator at the required height

Maintaining clear LiDAR visibility

Positioning the depth camera for eye-in-hand operation

Secure placement of detachable storage bins

Housing of the NUC while maintaining rover stability

These CAD resources allow full reconstruction, modification, or further development of the payload sled system.
